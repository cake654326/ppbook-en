# Set Up PPCom

#### Preparation
    
    cd ~/Documents/ppmessage/ppmessage/ppcom
    bower install
    cd ./gulp
    npm install 
    gulp
    
#### notice
When develope, run `gulp watch` firstly

    cd ~/Documents/ppmessage/ppmessage/ppcom/gulp
    gulp watch

When add new javascript files, update gulp/build.config.js
