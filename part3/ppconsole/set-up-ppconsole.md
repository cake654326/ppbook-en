# Set up PPConsole

#### Preparation
    
    cd ~/Documents/ppmessage/ppmessage/ppconsole
    bower install
    cd ./gulp
    npm install 
    gulp
    
#### Notice
When develope, run `gulp watch` firstly

    cd ~/Documents/ppmessage/ppmessage/ppconsole/gulp
    gulp watch

When add new javascript files, update gulp/build.config.js.
