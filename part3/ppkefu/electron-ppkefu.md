#### Developement
enter `~/Documents/ppmessage/ppmessage/ppkefu/ppkefu`.

* init `electron`

  install`electron-prebuilt, electron-packager, electron-builder`, run 

  ```
  npm run set-up-electron
  ```

* run electron

  after run gulp task, to run electron, type command

  ```
  npm run start
  ```

* generate installer

  ```
  npm run dist
  ```
